Game Verse

En esta API vas a poder obtener información de los videojuegos más populares

Endpoints:

    Videojuegos

  • Obtener todos los juegos

    GET /games

    Descripción: Obtiene una lista de todos los videojuegos registrados en la base de datos.

  • Buscar videojuego por nombre

    GET /games/nombre?name=nombre_del_juego

    Descripción: Permite buscar un videojuego por su nombre.

    Parámetro: name (nombre del videojuego)

  • Buscar videojuego por género

    GET /games/genero?genre=genero

    Descripción: Permite buscar un videojuegos por género.

    Parámetro: genre (genero del videojuego)

  • Crear videojuego

    POST /games/

    Descripción: Crea un nuevo videojuego en la base de datos.

    Esquema: { "name": "nombre del juego", "genre": "generos", "description": "Breve descripción", "developer": "Desarrollador del videojuego", "release_date": "fecha de lanzamiento", "platform": "Plataformas disponibles", "cover": "Imagen de portada" }

  • Actualizar un videojuego

    PUT /games/:id

    Descripción: Actualiza la información de un videojuego existente.

    Parámetro: ID (ID del juego a actualizar)

    Autenticación: Requiere Autenticación

  • Eliminar un videojuego

    DELETE /games/:id

    Descripción: Elimina un videojuego por su ID.

    Parámetro: ID (ID del videojuego a eliminar)

    Autenticación: Requiere Autenticación

    • Usuarios

  • Obtener todos los usuarios

    GET /users

    Descripción: Obtiene una lista de todos los usuarios registrados en la base de datos.

  • Crear un usuario

    POST /users

    Descripción: Crea un nuevo usuario en la base de datos.

    Esquema: { "name": "nombre del usuario", "email": "email del usuario", "password": "crear contraseña", "imagen": "imagen del usuario", }

  • Iniciar sesión

    POST /login

    Descripción: Obtiene una lista de todos los usuarios registrados en la base de datos.

    Esquema: { "email": "email del usuario registrado", "password": "contraseña registrada", }

  • Actualizar usuario

    PUT /users/:email

    Descripción: Actualiza la información de un usuario existente.

    Parámetro: Email (Email que quiera actualizar)

    Autenticación: Requiere Autenticación

  • Eliminar un videojuego

    DELETE /users/:id

    Descripción: Elimina un usuario por su ID.

    Parámetro: ID (ID del usuario a eliminar)

    Autenticación: Requiere Autenticación
